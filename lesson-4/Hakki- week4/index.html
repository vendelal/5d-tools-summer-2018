There are 3 ways of adding CSS to a project

Here they are in order of least specific to most specific:

  1. External file
    • This is a separate page from your HTML page.
    • This is by far the most common, and best practice way of adding CSS.
    • This is the method we'll be using all semester.
  2. Page specific
    • This method uses "style" tags at the beginning of an HTML document to add CSS specific to the page.
    • This type of CSS only applies to the page that it's written on.
    • It can be handy when you're sending someone an HTML file that you want to make sure has styles attached.
  3. Inline Styles
    • These are written in the same line that you add an element
    • The syntax looks like this: style="color: blue;"

How to make an external CSS file

  1. Create your HTML file
  2. Create your CSS file
  3. Save them in the same folder (or at least within the same root folder)
  4. Connect them with a link tag in the head of the HTML document
  5. Reload, edit, adjust, repeat!
