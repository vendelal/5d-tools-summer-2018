Utsa's Portfolio

This webpage will be a platform to showcase main projects I've worked on.

How the webpage will be divided into three pages:

  1. Absolut Craft:

    • This project is transforming packaging. The aim of the project was to make a new product for the brand Absolut in the food or cosmetics industry.
    • Absolut Craft is an infusion kit allows the user to create their own personalised flavours.
    • The page for this project will show concepts, moodboards, renders and final product images and branding.

  2. Absolut Challenge

    • Absolut Challenge is the reinvention of the game chess through the branding of Absolut.
    • The webpage for this product will explain the aim of the project followed by images of the final product.

  3. La Cascada

    • This project was to explore a space in Barcelona, Spain and create an exhibit representing the place.
    • La Cascada explores Parc de la Cuitadella to create a space development game as a souvenier for the park.
    • The page for this project will show concepts, history and finally the explanation and renders of the game/souvineer.
