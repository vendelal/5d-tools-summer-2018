My Portfolio Website

Website Description

My website is going to be a personal portfolio showcasing work related to my major: Computer Science and Design. Since I am going on co-op next year, a portfolio website is something that I've always wanted to set up for my future employers. I also want to have a personalized platform in which I can present all my projects in an aesthetically-pleasing and unique way.
